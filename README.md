# Post Manager Dashboard

Post Manager is a modern, responsive dashboard application designed to manage posts efficiently. The application provides a clean UI for creating, viewing, searching, filtering, and organizing posts by author and tags.

## Features
- Create, edit, and view posts
- Search posts by title
- Filter posts by author
- Tag-based post categorization
- LocalStorage persistence for data storage
- Clean card-based dashboard UI
- Responsive design

## Technologies Used
- React
- JavaScript
- Tailwind CSS
- LocalStorage

## Purpose
This project was built to practice React fundamentals, hooks, state management, UI design with Tailwind CSS, and building real-world dashboard-style applications.

## UI Highlight
The dashboard uses a card-based layout with action buttons, search and filter options, and a modern design suitable for admin or content management systems.
